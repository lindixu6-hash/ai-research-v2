# RESPONSES.md - 回答模板

> 根据意图类型选择对应的回答格式

---

## 1. Informational-Simple（简单信息型）

### 模板

```markdown
# 核心答案

[1-3句话直接回答问题，包含具体数据]

## 补充信息（可选）

[如果用户需要更多背景，可以展开2-3点]

---
**风格要求：**
- 开门见山，第一句就给答案
- 具体数字优于模糊描述
- 不需要复杂的结构化报告
- 不要"可能取决于"等废话
```

### 示例

```markdown
# 核心答案

美国公司的标准工作时间是 **上午9点到下午5点**，每周工作约40小时（周一至周五）。

## 补充信息

- **弹性工作制**：科技公司和初创企业常采用弹性工作时间
- **午餐时间**：通常有1小时午休（不计薪）
- **加班文化**：部分行业存在加班现象，但不如东亚普遍
```

## 2. Informational-Complex（复杂信息型）

### 模板

```markdown
# 核心答案

[直接结论，1-2句话]

## 关键点

1. **要点1**：[具体描述]
2. **要点2**：[具体描述]
3. **要点3**：[具体描述]

## 详细说明

[展开2-3段，提供背景和细节]

## 参考资源

- [资源1](链接)
- [资源2](链接)

---
**风格要求：**
- 结构清晰，分点明确
- 每个要点有具体内容
- 包含可操作的资源
```

---

## 3. Commercial（商业调研型）

### 模板

```markdown
# 核心答案

[一句话对比结论]

## 详细对比

| 维度 | 选项A | 选项B |
|------|-------|-------|
| 价格 | ... | ... |
| 性能 | ... | ... |
| 优缺点 | ... | ... |

## 选择建议

- **选择A如果**：[具体场景]
- **选择B如果**：[具体场景]

---
**风格要求：**
- 表格形式清晰对比
- 给出明确选择建议
- 不模棱两可
```

---

## 4. Research（研究综合型）

### 模板

```markdown
# 研究报告：[主题]

## 核心发现

[3-4句话总结核心发现]

## 关键数据

1. **数据1**：[具体数字 + 来源]
2. **数据2**：[具体数字 + 来源]
3. **数据3**：[具体数字 + 来源]

## 详细分析

### 背景
[背景信息]

### 核心问题
[关键问题分析]

### 趋势分析
[趋势和预测]

## 结论

[总结性判断和建议]

## 局限性

[说明数据的局限性和不确定性]

---
**风格要求：**
- 完整的研究报告结构
- 数据有来源标注
- 承认局限性
```

## 5. 通用禁止事项

### 绝对禁止的话术

- "可能取决于..."
- "有待观察..."
- "具体情况具体分析..."
- "从...角度来看..."
- "都有可能..."
- "不好说..."
- "包括但不限于..."

### 无关内容（根据类型过滤）

| 意图类型 | 禁止内容 |
|----------|----------|
| Informational-Simple | 学习路径、薪资就业 |
| Navigational | 详细分析、技能清单 |
| Transactional | 学习路径、技能发展 |
| ... | ... |
